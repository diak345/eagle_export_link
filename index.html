Export Symlink

Create symlinks for the currently selected items in Eagle.

Symlink type
Symlink points to the original file. Hard Link creates another file entry on the same drive.
How it works
  • Select items inside an Eagle folder.
  • Choose a destination folder when prompted.
  • If a folder is selected in Eagle, symlinks are created inside a subfolder with the same name.
  • If no folder is selected, symlinks are created directly in the export destination root.
Note: Hard links only work on the same drive. Symlinks may require permission depending on OS settings.
